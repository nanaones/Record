## AndroidEmulators in Windows 10

1. AndroidStudio
    - Price: Free
    - Performence: Low  
    - Useage 

        0. 기본 환경설정 완료
        1. AndroidStudio Start
        2. Android Virtual Device Manager(ADB) Start
        3. Type this
            ```
            $ expo start --android
            ```
            OR   
            ```
            $ yarn start --android
            ```
